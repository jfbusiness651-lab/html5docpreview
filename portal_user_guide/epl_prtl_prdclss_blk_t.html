Bulk Product Classification

This process explains how to classify a batch of poducts in a single batch upload.

Warning:
The results here are based on non-deterministic logic. The Product Classification feature is intended to provide you with information that you are responsible for verifying the correctness for your purposes. ePAL accepts no liability for incorrect results.
Tip:
You can download sample files from the UI by clicking the Download CSV Example or Download buttons.

Restrictions

The following restrictions apply:
  • You must use CSV or the WooCommerce format.
The following restrictions apply to the CSV file:
  • The header row must contain 2 headers: description and hscode.
  • You can specify values for either header.
  • Values must be at least 2 characters long.
