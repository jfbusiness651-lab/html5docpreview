Providers UI

Use this UI to configure your tax, customs and shipping providers.

To access the UI, click Providers. The following UI is displayed:

You can select a provider for each of the following:
Table 1. Providers
Provider Description
Tax Provider

Choose the source for tax rate calculations from one of the following:

  • ePAL default
  • Hurricane
  • Vertex Inc
Customs Provider Choose the source for Customs duty calculations from one of the following:
  • ePAL default
  • Hurricane
Shipping Provider Choose the shipping provider from one of the following:
  • ePAL default
  • Shippo
