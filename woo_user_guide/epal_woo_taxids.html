B2B and Tax ID Validation

ePAL uses Tax IDs such as VAT IDs to help validate B2B transactions.

A Tax ID is a value that identifies a party as a participant in a particular Tax Scheme. For example, a VAT ID identifies a party as a participant in a VAT scheme.

ePAL uses these in cross-border B2B transactions to help ensure that the Tax IDs your customers provide are valid.

ePAL will check if a Tax ID is valid in the VIES database and will return an associated address when available.
