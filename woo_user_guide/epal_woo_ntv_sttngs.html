Reference Guide

This guide serves as a reference that lists all the relevants UIs and other resources.

Other settings are available in the ePAL portal. More information is provided in the LINK topic.

WooCommerce UIs

The following settings are available in the WooCommerce UI:
  • Shipping Address Settings
  • ePAL Integration Settings UI

ePAL Portal

  • ePAL Tax Calculation UI
    You can use the ePAL Tax Calculation UI to confure how ePAL interacts with your tax settings in WooCommerce
  • Shipping Address Settings
    Use the Shipping settings in WooCommerce to create a custom shipping address.
  • ePAL Integration Settings UI
    Use this UI to configure the integration with your ePAL account.
  • ePAL Tax Configuration UI
    Use this UI to manage all your ePAL settings.
