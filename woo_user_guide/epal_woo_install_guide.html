Installation Guide

This guide provides you with all the information that you need to install ePAL on WooCommerce.

This topic will guide you through downloading, installing and configuring the plugin. More general information about the plugin and its features is provided in the Getting Started with the ePAL WooCommerce Plugin topic.

Installation Steps

To start using ePAL in your checkout, complete the following steps:
  1. Review the Prequisites and Minimum Requirements sections of this topic.
  2. Sign up for ePAL at http://3.149.180.94/sign-up.
  3. Retrieve and upload the Woo Commerce plugin as described in the Uploading the ePAL Plugin topic.
  4. Install the ePAL plugin for Woo Commerce. The plugin includes a guided installation to help you set it up. This is explained in the Installing the ePAL Plugin topic.
  5. Test the installation as described in the Testing the Plugin topic.

Prerequisites

Before you begin the installation, consider the following prerequisites:
  • If you have a Tax ID, you need to read about the type of Tax IDs that are compatible with the service. More information is provided in the LINK topic.
  • If you are outside the EU, you may want to consider using IOSS. More information is provided in the LINK topic.
  • You can sign up for ePAL at http://3.149.180.94/sign-up.
