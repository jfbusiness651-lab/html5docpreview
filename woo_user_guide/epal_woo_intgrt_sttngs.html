ePAL Integration Settings UI

Use this UI to configure the integration with your ePAL account.

You can open this UI from step 2 in the guided installation. This is explained in the Installing the ePAL Plugin topic.

You can also open the UI by opening WooCommerce and navigating to WooCommerce > ePAL > Configure. The following UI is displayed:
